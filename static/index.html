How to Export the CSV from Your Anti-DDoS Dashboard

The Traffic Analysis module accepts NetFlow/event CSV files exported directly from your Anti-DDoS appliance (e.g. Fortinet FortiDDoS). Follow these steps:

1
Log in to your Anti-DDoS management dashboard and navigate to Event Traffic or the equivalent traffic graph view.
2
Select your desired time range and traffic filters (protocol, interface, etc.).
3
Click the download icon (top-right corner of the chart) to export the data as a CSV file.
4
Upload the downloaded .csv file here in the Traffic Analysis tab.

IP Reputation Scoring

Each IP is checked against up to 4 threat intelligence sources. Points from all sources are summed into a final score (0–100).

Hash Analysis Scoring

File hashes (MD5, SHA-1, SHA-256) are submitted to VirusTotal. The verdict is determined by how many AV engines flag the file.

Verdict Condition Meaning
MALICIOUS ≥ 5 engines flag Widely detected malware/threat file.
HIGH RISK 2 – 4 engines flag Detected by a minority of engines. Likely malicious.
SUSPICIOUS 1 engine flags Single detection — could be a false positive.
CLEAN 0 engines flag No detections. File appears benign.
⚠️ Not Found — If VT has never seen the hash, the result is NOT FOUND. This does not mean the file is safe — it may be novel or rare malware not yet submitted to VirusTotal.

Traffic Analysis Scoring

NetFlow/CSV traffic is analyzed for attack patterns. Each detected indicator adds to the attack score (0–100). IP reputation from auto-investigation adds a bonus on top.

Attack Indicators & Points
Severity Indicator Points Description
HIGH SYN Flood +30 >70% of flows are pure SYN — classic volumetric DDoS.
HIGH Layer 7 HTTP Flood +25 PSH+ACK dominates flows — active HTTP application flood.
HIGH Amplification / Reflection +25 Avg packet size >900B across most flows (DNS/NTP/SSDP abuse).
HIGH Anomalous Flow Size +20 Many individual flows exceed 50 MB — abnormal for legitimate sessions.
MEDIUM Traffic Concentration +15 Single source contributes >60% of total bytes — abnormal dominance.
MEDIUM UDP Flood +15 >70% UDP flows with high packet rate — volumetric UDP attack.
LOW Non-standard Ports +8 Traffic to unusual destination ports — possible port scanning or C2.
